Preguntas del test

Aquí se debe indicar los enunciados de las preguntas del test y el estilo de aprendizaje al que pertenece cada una de ellas. Además, para cada pregunta se debe indicar la puntuación obtenida según se marque o no cada una de las respuestas posibles. Así, para cada una de las respuestas, aparecen debajo dos valores: el de la izquierda para indicar la puntuación obtenida si no se elige dicha respuesta y el de la derecha para la puntuación obtenida en caso de que sí se elija dicha respuesta.
